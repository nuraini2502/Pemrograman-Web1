Nama saya Nuraini, saya anak pertama dari 2 bersaudara saya lahir di Jakarta, 25 Februari 2004. Saya berada di Jakarta sejak saya lahir, sampai saat ini saya masih di jakarta. Cita -cita saya ingim membahagiakan kedua orang tua saya.

Lalu saya lanjut ke MTS Salafiyah Syafi'iyyah selama 3 tahun, alasan saya memilih MTS Salafi'iyyah Syafiiyyah karena jarak yang di tempuh ke sekolah sangat dekat dan tidak perlu mengeluarkan ongkos. Setelah 3 tahun saya melanjutkan pendidikan ke SMK Salafiyah Syafi'iyyah. Kalau ini saya tidak memiliki alasan untuk masuk SMK tersebut karena orang tua saya meminta saya melanjutkan di sana saja, tetapi saya banggan karna saya bisa masuk SMK Salafiyah Syafi'iyyah karena saya dapat merasakan perbedaaan dari sekolah lain lewat cerita teman saya.
